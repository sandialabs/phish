set command

Syntax:

set keyword value ...

  • one or more keyword/value pairs can be appended 
    possible keywords = none are yet defined

Examples:

No set keywords are yet defined.

Description:

Set is a command that can be used in a PHISH input script which is recognized by the bait.py setup program. It (re)sets values that are used by the bait.py program as it reads and processes commands from the PHISH input script.
